Naming conventions for GraphDB repositories
Import files as named graphs and use the URI as both the Base IRI and the Target graph name

UFOKN-Basic
	Files:
		ufokn-core.ttl
		ufokn-geo.ttl
		ufokn-utility-connection.ttl
		dkk-basic-dataset-01.ttl
	Graphs (use URI as both Base IRI and Target graph name):
		http://schema.ufokn.org/core/v2/
		http://schema.ufokn.org/geo/v2/
		http://schema.ufokn.org/utility-connection/v2/
		http://schema.ufokn.org/example/dkk-basic-dataset-01
